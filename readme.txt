Display Easy Digital Downloads categories with images in a grid layout using a shortcode.

== Description ==

This WordPress plugin allows you to easily display Easy Digital Downloads (EDD) categories with their associated images in a grid layout on your website. Simply use the provided shortcode to add the category display to any post, page, or widget area.

== Installation ==

1. Upload the `easy-digital-downloads-categories-display` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the `[edd_categories]` shortcode to display the EDD categories with images on your website.

== Frequently Asked Questions ==

= Can I customize the styles of the category display? =

Yes, you can customize the styles of the category display by adding custom CSS to your theme's stylesheet or using a custom CSS plugin.
